Installazione del BAT Mod

Il BAT Mod, a differenza del BUG Mod, non può essere installato come mod principale (nella cartella '..\My Games\Beyond the Sword'), ma solamente come mod opzionale, nella cartella '..\My Games\Beyond the Sword\Mods\BUG Mod'.
Pertanto, non sarà possibile utilizzare contemporaneamente un altro mod come mod opzionale (ma sarà possibile utilizzare un altro mod come mod principale) e tutti gli altri giocatori che vogliono giocare con voi in Multiplayer o in un Succession Game dovranno avere installata la vostra stessa versione del BAT Mod.


Installazione automatica (file .exe)

  1. Disinstallate le precedenti versioni del BAT Mod (e verificate che effettivamente non sia rimasta traccia dei vecchi file nella cartella '..\My Games\Beyond the Sword'). Attenzione! Non farlo potrebbe comportare vari malfunzionamenti del BAT Mod.
  2. Lanciate il file 'BATMod_xxx.exe'.
  3. Quando vi viene chiesta la directory di installazione, controllate che quella indicata sia veramente la cartella di Civ4:BtS all’interno della cartella Documenti (solitamente '...\My Games\Beyond the Sword'), altrimenti sfogliate le vostre cartelle fino a raggiungerla.

Installazione manuale (file .zip)

  1. Individuate la cartella di Civ4:BtS all’interno della cartella Documenti (solitamente 'Documenti\My Games\Beyond the Sword').
  2. Cancellare tutte le tracce delle precedenti versioni del BAT Mod. Attenzione! Non farlo potrebbe comportare vari malfunzionamenti del BAT Mod.
  3. Dezippate il file 'BATMod_xxx.zip' all’interno della cartella 'Beyond the Sword', avendo cura di conservare i percorsi impostati nel file zip.
