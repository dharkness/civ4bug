How to Install the BAT Mod

The BAT Mod, differently from the BUG Mod, can't be installed as main mod (in the '..\My Games\Beyond the Sword' folder), but only as optional mod, in the '..\My Games\Beyond the Sword\Mods\BUG Mod' folder.
So, you can't use it together with another optional mod (but you can use it together with another mod which is installed as main mod) and all the players who wish to play together with you in a multiplayer or Succession game must have the same version of the BAT Mod installed.


Instructions for the Self-Extracting Setup (.exe file)

  1. Uninstall older versions of BAT Mod (and check that all traces of old BAT files in the '..\My Games\Beyond the Sword' folder have been removed). Warning! Not doing so may cause the BAT Mod not to work properly.
  2. Run the 'BATMod_xxx.exe' file.
  3. When the setup asks for an install directory, choose the Civ4:BtS folder inside your My Documents folder (usually '...\My Documents\My Games\Beyond the Sword').

Instructions for the Manual Setup (.zip file)

  1. Locate the Civ4:BtS folder inside your My Documents folder (usually 'My Documents\My Games\Beyond the Sword').
  2. Delete all traces of older versions of the BAT Mod. Warning! Not doing so may cause the BAT Mod not to work properly.
  3. Unzip the 'BATMod_xxx.zip' file inside the 'Beyond the Sword' folder, preserving the paths set inside the zip file.
