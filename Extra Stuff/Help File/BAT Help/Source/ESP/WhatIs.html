What is The BAT Mod?

The BtS Artistic Tweaks (BAT Mod) is a modification for Civilization IV: Beyond the Sword.

Civilization IV was designed to be extremely easy to modify, but not all modifications are compatible with each other.
In an effort to combine these many different mods from the Civilization Community into one concise program, the BUG Project was born.

The BAT Mod is an offshot of the original BUG Mod, including all the graphic and text changes (in particular those which allow to make each civilization as unique as they are in reality), which are not included into the BUG Mod due to their big size.
On the other hand, to make the install process as easy as possible, the BUG Team includes into the BAT Mod setup file also the full BUG Mod pack, thanks to its small size.

The BAT Mod combines mods that do not change the basic game play of Civ4 while enhancing its look and easy of use. Also, any additions or changes made will not provide any information that cannot be figured out within the game itself.
We feel this mod is safe for use in any Succession Game, but all players will have to run it from the '..\My Games\Beyond the Sword\Mods' folder. If you find any changes within the code that modify gameplay, please contact us, and we will make it our priority to fix the problem.

The best way to understand the BAT Mod, anyway, is to download the latest version and to try it.
