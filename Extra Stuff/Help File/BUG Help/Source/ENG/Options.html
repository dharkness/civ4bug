Options Screen

The core to The BUG Mod is the ability to pick and choose the way your game looks and plays. Once the game starts you can customize the mod by hitting Ctrl + Alt + O. From this screen you can adjust the look and feel of many of the features within the game. Below you can find out more about each of the Options Tabs.

 

General - General Options, involving the Main Interface and City Screen.

Advisors - Options to change settings for the Advisors and the Sevopedia.

Clock - Settings to adjust the Clock Display.

Scoreboard - Advanced Scoreboard Options.

Alerts - Set up which alerts are displayed at the start of each turn.

Logging - Setting for the Autolog.

Unit Naming - Options for setting up Unit Naming for new unit builds.

PLE - Options for configuring the way units icons are displayed, and filtering options.

 

 

Note: The Asioasioasio’s Wide City bar mod can’t be removed using the Option Screen (Alt-Ctrl-O). If you want to remove it, simply delete the "CustomAssets\Art\Interface\Citybar" folder.
