Plot List Enhancement Options

 

Show Mode/Filter Buttons: When Checked this option will show PLE Buttons.

Default View: How units are displayed on the screen. Options are Single Row, Multiple Rows, Vertical Groups, and Horiztonal Groups.

Default Grouping: Option to change the way units are grouped together in Vertical and Horiztonal Grouping. Options are Unit Type, and Selection Group.

Filter Behavior: Either default PLE filtering behavior, or the BUG filtering Method.

Move Highlighter: Highlights the moves a unit can make when you hover the mouse over the unit, while holding the [ALT].

 

Indicators:

Wounded Dot: Show a healthy or wounded symbol next to the unit icon.

Great General: Places a star next to units that are lead by a Great General.

Promotion Avaliable: Places a light blue border around units that can be promoted.

Upgrade Avaliable: Places an orange up arrow on the lower-right corner of units that can be upgrade, ignoring cost.

Mission Tag: Shows an icon indicating what mission, or action, the unit is currently preforming.

 

Health Bar: When selected will show a bar indicating the units health level. You can configure the Healthy and Wounded colors you prefer, and can hide during combat.

Movement Bar: When selected will show a bar indicating the units current movement points. You can change the colors used for Unmoved, Has Moved, and Cannot Move.

Unit Info Tooltip: This is for the Hover text that will appear when you hover your mouse over a unit's icon.

Unit Name: The color the units name are displayed in.

Upgrade Cost: Displays the cost of a units upgrade.

Can Afford: Change the color indicating if you can upgrade that unit.

Cannot Afford: Change the color indiciating if you cannot upgrade that unit.

Unit Specialites: Displays the special bonuses a unit has.

Unit Type: Color to display bonuses based on the units type.

Promotion: Color to display bonuses based on promotion that unit has undergone.
