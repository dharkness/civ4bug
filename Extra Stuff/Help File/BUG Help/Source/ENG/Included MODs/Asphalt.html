I Love Asphalt! by ricardojahns

This modification will make roads during the Industrial, Modern, and Future Era, darker so it is much easier to tell the difference between Roads and Railroads. If you would like to remove simply delete the directory "Art\Interface\Terrain\Routs\Modern Roads" and the file "XML\Art\CIV4RouteModelInfos.xml".
