Installation

WARNING: The BUG Project may overwrite some or all of the files in your current "CustomAssets" folder. If you wish to keep your current "CustomAssets" folder you have two options:

1) Backup your current "CustomAssets" folder and use a merge program to merge the two directories. This is a painstaking process. If you go this route be warned that your game may not work if not done properly.

2) If you are using the Setup file, change the Install Directory to "Mods\BUG Mod", install the mod, and rename the "CustomAssets" folder inside it to "Assets". If you are using the ZIP file, then extract to "Mods\BUG Mod", and rename the "CustomAssets" folder inside it to "Assets". If this is done, you will have to load BUG for each game, and will require others who you are playing in Multi-Player, or in a Succession Game to have the same version of the BUG Mod loaded as well.

If you use the second method and don't want to have to manually load the mod every time you launch the game, you can change the "Mods" parameter in your "CivilizationIV.ini" file from "0" to "BUG Mod". This tells Civ4 to load BUG Mod automatically. To use a different mod (or no mod), simply click "UNLOAD MOD" in the main menu.



Self-Extracting .EXE instructions:

1) Uninstall or delete all traces of older version of BUG Mod. Renaming directories to something else to backup your files, will work as well.** IF YOU DO NOT DO THIS THE MOD WILL NOT WORK PROPERLY!**

2) Run "BUGMod_xxx.exe" file

3) Choose which of the listed install option you would like to configure The BUG Mod with.

4) Restart Civ4.



ZIP File instructions:

1) Locate your CIV:BtS folder (Usually within "My Documents\My Games\Beyond the Sword").

2) Uninstall or delete all traces of older version of BUG Mod. ** IF YOU DO NOT DO THIS THE MOD WILL NOT WORK PROPERLY!**

3) Copy contents of "BUGMod_xxx.zip" into your CIV:BtS folder. Make sure "BUG Mod.ini" is placed in the "My Documents\My Games\Beyond the Sword\BUG Mod " folder, not the CustomAssets folder. If installed as a Mod, ensure that there is a copy of "BUG Mod.ini" is in the "Mods\BUG Mod" folder. Only name this directory "BUG Mod."

4) Restart Civ4.
