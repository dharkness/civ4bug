Installation

WARNING: The BUG Project may overwrite some or all of the files in your current "CustomAssets" folder. It is recommended to back up this folder if you have other items you wish to preserve, not related to BUG.

 

Self-Extracting .EXE instructions:

1) Uninstall or delete all traces of older version of BUG Mod. Renaming directories to something else to backup your files, will work as well.** IF YOU DO NOT DO THIS THE MOD WILL NOT WORK PROPERLY!**

2) Run "BUGMod_xx.exe" file

3) Choose which of the listed install option you would like to configure The BUG Mod with.

4) Restart Civ4.



ZIP File instructions:

1) Locate your CIV:BtS folder (Usually within "My Documents\My Games\Beyond the Sword").

2) Uninstall or delete all traces of older version of BUG Mod. ** IF YOU DO NOT DO THIS, THE MOD WILL NOT WORK PROPERLY!**

3) Copy contents of "BUGMod_xx.zip" into your CIV:BtS folder. Make sure "BUG Mod.ini" is placed in the "My Documents\My Games\Beyond the Sword\BUG Mod " folder, not the CustomAssets folder. If installed as a Mod, ensure that there is a copy of "BUG Mod.ini" is in the "Mods\BUG Mod" folder. Only name this directory "BUG Mod."

4) Restart Civ4.
