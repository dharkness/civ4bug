Tastaturkürzel der BUG Mod

Nachfolgend finden Sie die Tastenkombinationen, die Sie benötigen werden, um die BUG Mod effektiv zu nutzen. Es gibt nicht viele, so dass sie relativ einfach zu merken sein sollten.

  • Strg + Alt + O - Optionsbildschirm von BUG.

  • Alt + M oder Strg + Alt + R - Eine Erinnerung schreiben.

  • Alt + L - AutoLog aufrufen.

  • Alt + B - Kampfstatistik erstellen.

  • Alt + E - Einen Eintrag ins AutoLog schreiben.
