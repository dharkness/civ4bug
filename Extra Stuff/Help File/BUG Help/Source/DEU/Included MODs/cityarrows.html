City Cycle Arrows von SimCutie

 

Über City Cycle Arrows

Diese Mod erlaubt das Durchschalten der eigenen Städte im Hauptbildschirm.

 

Anleitung

Über der eigenen Flagge im Hauptbildschirm werden zwei weiße Pfeile angezeigt. Durch einen Klick auf einen dieser Pfeile erscheinen Informationen der nächstgelegenen Stadt (ohne detaillierte Bügerstatistiken) im Hauptbildschirm.

Im Optionsbildschirm der BUG Mod unter "Allgemein" kann die Anzeige an- oder abgeschaltet werden.
