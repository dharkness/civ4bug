Custom Domestic Advisor Pages (Version 2.6) von Cammagno

 

Diese Mod ist ein Zusatz zum Custom Domestic Advisor (CDA). Dadurch wird jede verfügbare Information auf verschiedenen Seiten angeordnet, durch die sehr einfach geblättert und eine gesuchte Information schnell gefunden werden kann.

Diese Version von Cammagno’s CDA Pages wurde exklusiv für die Verwendung mit der BUG Mod geschrieben. Falls Cammagno's CDA Pages nicht verwendet werden sollen, kann die Datei "CustomDomAdv.txt" im Verzeichnis "CustomDomAdv" gelöscht werden. CDA erstellt beim nächsten Spielstart eine Standard-Datei, die nach eigenem Belieben verändert werden kann.

Nachfolgend finden Sie eine Auflistung aller Seiten zusammen mit den verwendeten Spalten jeder Seite und deren Standard-Breite.
