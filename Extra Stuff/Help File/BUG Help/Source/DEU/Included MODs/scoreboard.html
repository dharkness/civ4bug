Advanced Scoreboard von EmperorFool

 

Über Advanced Scoreboard

Diese Mod stellt eine vollständig anpassbare Punkteanzeige als Teil der BUG Mod bereit. Dabei wurde sowohl TheLopez' "Dead Civ Scoreboard" Mod als auch Porges "Attitude Icons" integriert (beide sind immer noch optional). So kann sehr genau festgelegt werden, was angezeigt wird und in welcher Reihenfolge. Außerdem wurde eine Option hinzugefügt, um die gerade erforschten Technologien gegnerischer Zivilisationen als Symbol statt als Text anzuzeigen, wenn diese bekannt sind.

Dead Civ Scoreboard - entfernt oder graut Zivilisationen aus, die während einer Partie ausgelöscht wurden.

Attitude Icons - fügt der Punkteanzeige Smilies hinzu, die die momentane Einstellung gegnerischer Zivilisationen gegenüber der eigenen repräsentieren.

Power Ratings - zeigt das Verhältnis der Macht der eigenen gegenüber einer gegnerischen Zivilisationen an. Je höher die Zahl ist, desto stärker ist die eigene im Verhältnis zu einer gegnerischen Zivilisation.

 

Anleitung

Dies Mod kann im Optionsbildschirm der BUG Mod (Strg-Alt-O) unter "Punkteanzeige" angepasst werden. Die folgenden Optionen befinden sich in der rechten Spalte unter "Erweitertes Layout". Sie werden mit der Option "Aktivieren" im Spiel wirksam, so dass das Aussehen der Punkteanzeige angepasst werden kann.

Im Textfeld "Reihenfolge der Spalten" kann gewählt werden, was angezeigt werden soll und in welcher Reihenfolge. Jedes Element wird durch einen einzelnen und einzigartigen Buchstaben repräsentiert. Im Tooltip des Textfeldes werden alle Elemente detailliert erklärt:

  • S - Wertung einer Zivilisation

  • C - Name einer Zivilisation/eines Oberhaupts (siehe Option für "Anzeigename")

  • ? - Zivilisation noch nicht kontaktiert (zeigt ein ? für noch nicht kontaktierte Teammitglieder anderer Teams)

  • W - Kriegszustand oder Friedensvertrag mit einer Zivilisation (zeigt gekreuzte Schwerter oder ein "Peace"-Zeichen an)

  • E - Gutes Spionagepunktverhältnis gegenüber einer Zivilisation (zeigt ein Spionsymbol an)

  • P - Machtverhältnis zwischen der eigenen und einer gegnerischen Zivilisation

  • T - Derzeitiges Forschungsprojekt einer Zivilisation (gilt für Vasallen, Verbündete und durch Spionage erlangt)

  • U - Anzahl verbleibender Runden bis zur Erforschung (siehe T oben)

  • N - Handelsverbindungen mit einer Zivilisation (zeigt das Symbol mit den drei Pfeilen an)

  • B - "Offene Grenzen" mit einer Zivilisation (zeigt das Schriftrollensymbol an)

  • D - "Verteidigungsbündis" mit einer Zivilisation (zeigt ein ähnliches Symbol wie bei B an)

  • R - Staatsreligion einer Zivilisation

  • A - Einstellung einer Zivilisation gegenüber der eigenen (zeigt einen Smiley an)

  • H - Erzfeind der eigenen Zivilisation (zeigt einen wütenden Bürger an)

  • * - Warten auf den Zug einer Zivilisation

  • L - Netzwerkstatistik einer Zivilisation (Ping)

  • O - Verbundener Spieler ist "out-of-sync"

Diese Elemente werden nur für Gegner angezeigt.

Diese Elemente werden nur in Multiplayer-Partien angezeigt.

 

Zusätzliche Optionen:

  • Die Option "Linksbündige Namen" richtet die Namen von Zivilisationen/Oberhäuptern nach links statt nach rechts aus.

  • Die Option "Technologiesymbole" zeigt die Symbole statt der Namen der Technologien an, die andere Zivilisationen momentan erforschen.
