Über Improved Unit Icons

Diese Mod verbessert die Darstellung der Einheitensymbole und fügt ihnen zusätzliche Informationen hinzu. Es ist eine leicht abgespeckte Version der Plot List Enhancement Mod von 12Monkeys.

Den Symbolen werden zwei neue Features hinzugefügt:

1) Ein Einheitensymbol erhält einen blauen Rahmen, wenn die Einheit befördert werden kann.

2) Den Einheitensymbolen wird die gerade ausgeführte Aktion als Text hinzugefügt. Das heißt, wenn eine Einheit verschanzt ist, wird am unteren Rand des Symbols das Wort "FORT" (engl. fortified) erscheinen, bei automatisierten Arbeitstrupps entsprechend "AUTO" etc.

 

Anleitung

Jedes dieser Features kann im Optionsbildschirm der BUG Mod unter "Allgemein" an- oder abgeschaltet werden.Die Optionen "Beförderung verfügbar" oder "Einheitenaktionen" deaktivieren die entsprechenden Features.
