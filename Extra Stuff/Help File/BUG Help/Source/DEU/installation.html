Installation

ACHTUNG: Die BUG Mod überschreibt einige oder alle Dateien, die sich im Verzeichnis "CustomAssets" befinden. Falls dies nicht beabsichtigt ist, gibt es zwei Möglichkeiten:

1) Zunächst sollte von dem bereits vorhandenen Verzeichnis "CustomAssets" eine Sicherungskopie erstellt und dann ein Programm verwendet werden, um beide Verzeichnisse zu verschmelzen. Das ist ein mühsamer Prozess, der, falls er nicht richtig vorgenommen wird, dazu führen kann, dass das Spiel danach nicht mehr funktioniert.

2) Bei Verwendung der Setup-Datei muss das Installationsverzeichnis in "Mods\BUG Mod" geändert werden. Bei Verwendung der ZIP-Datei muss diese manuell in das Verzeichnis "Mods\BUG Mod" entpackt werden. In beiden Fällen muss das im Verzeichnis "BUG Mod" befindliche Verzeichnis "CustomAssets" in "Assets" umbenannt werden. Danach kann BUG wie jede andere Mod separat geladen werden.

Bei der zweiten Methode muss BUG bei jedem Spielstart manuell geladen werden. Falls das nicht gewünscht wird, kann der Parameter "Mods" in der Datei "CivilizationIV.ini" von "0" in "BUG Mod" geändert werden. Eine weitere Möglichkeit ist, das Spiel mit dem Paramter "-mod= BUG Mod" zu starten. In beiden Fällen wird die BUG Mod automatisch geladen. Um eine andere (oder keine) Mod zu verwenden, muss im Hauptmenü der Menüpunkt "MOD ENTLADEN" gewählt werden.



Anweisungen für die Selbstextrahierende Datei:

1) Deinstallieren oder löschen Sie alle Spuren von alten Versionen der BUG Mod. ** FALLS DIESER SCHRITT NICHT DURCHGEFÜHRT WIRD, KANN ES PASSIEREN, DASS DIE MOD MÖGLICHERWEISE NICHT RICHTIG FUNKTIONIERT! **
2) Die Datei "BUGMod_xxx.exe" ausführen.
3) Wählen Sie unter den angezeigten Installationsoptionen diejenige aus, mit der Sie die BUG Mod installieren möchten.
4) Starten Sie nach der Installation Civ4.



Anweisungen für die ZIP-Datei:

1) Finden Sie das BtS-Verzeichnis (normalerweise im Verzeichnis "Eigene Dateien\My Games\Beyond the Sword").
2) Deinstallieren oder löschen Sie alle Spuren von alten Versionen der BUG Mod. ** FALLS DIESER SCHRITT NICHT DURCHGEFÜHRT WIRD, KANN ES PASSIEREN, DASS DIE MOD MÖGLICHERWEISE NICHT RICHTIG FUNKTIONIERT! **
3) Kopieren Sie den Inhalt der Datei "BUGMod_xxx.zip" in das BtS-Verzeichnis. Stellen Sie sicher, dass sich die Datei "BUG Mod.ini" im Verzeichnis "Beyond the Sword" befindet und nicht im Verzeichnis "CustomAssets". Falls BUG als normale Mod installiert wird, sollte sich die Datei "BUG Mod.ini" im Verzeichnis "Mods\BUG Mod" befinden.
4) Starten Sie nach der Installation Civ4.
