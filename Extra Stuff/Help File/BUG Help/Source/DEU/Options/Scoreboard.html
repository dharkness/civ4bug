Punkteanzeige

 

Anzeigename:

Bestimmt welcher Name für eine Zivilisation in der Punkteanzeige verwendet wird. Entweder der Name des Oberhaupts, der Zivilisation oder beides zugleich.

Einstellung als Smiley:

Zeigt die Einstellung jedes Computerspielers gegenüber der eigenen Zivilisation als Smiley an.

 

Ausgelöschte Zivilisationen:

Ausgelöschte Zivilisationen können auf verschiedene Arten angezeigt werden (oder auch gar nicht). Außerdem können diese mit dem Attribut "tot" versehen oder ihr Name ausgegraut werden. Diese Optionen können auch miteinander kombiniert werden.

 

Machtverhältnis

Zeigt die Machtverteilung zwischen der eigenen und den anderen Zivilisationen als Verhältniswert an, falls genug Spionagepunkte investiert wurden.

Standardfarbe:

Bestimmt die Standardfarbe, in der das Machtverhältnis erscheint.

Gutes Verhältnis:

Bestimmt die Prozentzahl, ab der das Machtverhältnis von ausgeglichen in gut umschlägt.

Farbe (gut):

Bestimmt die Farbe, in der gute Machtverhältnisse erscheinen.

Schlechtes Verhältnis:

Bestimmt die Prozentzahl, ab der das Machtverhältnis von ausgeglichen in schlecht umschlägt.

Farbe (schlecht):

Bestimmt die Farbe, in der schlechte Machtverhältnisse erscheinen.

 

Erweitertes Layout

Stellt zusätzliche Optionen für das Layout der Punkteanzeige zur Verfügung, mit denen diese verändert werden kann.

Reihenfolge der Spalten:

Bestimmt, welche Informationen die Punkteanzeige bereithält. Außerdem kann auch die Reihenfolge der angezeigten Informationen geändert werden. Die verfügbaren Optionen sind:

  • S - Wertung einer Zivilisation

  • C - Name einer Zivilisation/eines Oberhaupts

  • ? - Zivilisation noch nicht kontaktiert

  • W - Kriegszustand oder Friedensvertrag mit einer Zivilisation

  • E - Gutes Spionagepunktverhältnis gegenüber einer Zivilisation

  • P - Machtverhältnis zwischen der eigenen und einer gegnerischen Zivilisation

  • T - Derzeitiges Forschungsprojekt einer Zivilisation (gilt für Vasallen, Verbündete und durch Spionage erlangt)

  • U - Anzahl verbleibender Runden bis zur Erforschung

  • N - Handelsverbindungen mit einer Zivilisation

  • B - "Offene Grenzen" mit einer Zivilisation

  • D - "Verteidigungsbündis" mit einer Zivilisation

  • R - Staatsreligion einer Zivilisation

  • A - Einstellung einer Zivilisation gegenüber der eigenen

  • H - Erzfeind der eigenen Zivilisation

  • * - Warten auf den Zug einer Zivilisation

  • L - Netzwerkstatistik einer Zivilisation (Ping)

  • O - Verbundener Spieler ist "out-of-sync"

Linksbündige Namen:

Zeigt den Namen von Zivilisationen/Oberhäuptern linksbündig statt rechtsbündig an.

Technologiesymbole:

Zeigt die Symbole statt der Namen der Technologien an, die andere Zivilisationen momentan erforschen, falls diese bekannt sind.
