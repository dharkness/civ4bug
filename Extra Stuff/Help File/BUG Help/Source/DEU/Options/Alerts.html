Warnungen

 

Erinnerungen aktivieren:

Ermöglicht das Schreiben von Erinnerungsmeldungen (Tastenkombination [ALT-M]).

Mit Autolog protokollieren:

Protokolliert die Erstellung und Auslösung von Erinnerungen (verwendet "eotinbs Autolog" Mod).

Civ4lerts aktivieren:

Lässt Warnungen erscheinen, die bevorstehende oder bereits eingetretene Ereignisse melden.

 

** Sie finden eine Beschreibung aller Warnungen unter Civ4lerts in dieser Hilfe-Datei. **
