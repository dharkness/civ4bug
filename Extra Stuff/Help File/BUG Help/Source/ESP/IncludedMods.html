Included Mods

The BUG Project would no exist if not for the hard work already started by the rest of the Civilization Modding Community. We have included the original Read Me's of these Mods within this help file. If we have change the original Mod in anyway, we have tried to document it, but sometimes things can slip through the cracks! If you feel more information is needed, we left something or someone out, or if you have any other suggestions, then please Contact Us.
