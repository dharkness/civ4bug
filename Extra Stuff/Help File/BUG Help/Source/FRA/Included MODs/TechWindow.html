TechWindow

by Roamty


TechWindow enhances the screen that pops up when a new tech is researched. All newly available units, buildings and abilities are shown separately to give the screen a more clearer layout. Furthermore the screen will now pop up when you traded a tech with another civ and when you get a tech from a tribal village. Other new features include the display of the whole Civilopedia entry of a newly acquired tech below the tech's quote and the option to display the tech splash screen in a wider version.


Instructions

This mod can be enabled in the General tab of the BUG Options Screen (Alt+Ctrl+O) by simple selection of the 'View' checkbox.
Here you can chose which view to display when you discover a technology and if you want the the Civilopedia text to be included after the quote for the discovered technology.
