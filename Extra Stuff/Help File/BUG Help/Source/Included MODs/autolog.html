Autologger by eotinb

 

About Autologger

A mod to facilitate note taking, with succession games in mind.

 

Instructions

The autologger can be configured through the BUG Options screen. The Default directory where the logs are stores is in the My Docs\My Games\Beyond the Sword\Autolog directory. The default file name is the name you used as your Leader name. You can change these from within the BUG Options screen. The shortcut commands you should know for the autologger are:

  • Alt + L - Toggle autoLogging

  • Alt + B - Create Battle dump

  • Alt + E - Add custom Entry to autologger.

You can also change the format used. The options are HTML, Plain Text, Forum Codes, and Forum Codes with color. Default is Forum Codes, which is designed to be copy and pasted straight to a forum such as CFC, and posted to your story or succession games.

One word of warning. Using of autologger can be spammy, and take much of the heart of writing up a succession game report. A suggested method of using it is to write a summary of what happened, with many screen shots, and embedding the Autolog dump into a Spoilers tab within your message. Refer to the documentation on the forum you use on how to do this.
