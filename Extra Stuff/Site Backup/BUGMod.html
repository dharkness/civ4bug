The BUG Mod

Home The BUG Mod The BAT Mod Forum Blue Marble The BUG Blog

Civilization IV: Beyond the Sword has the ability to be modified through Python Code, as well as XML code.  The Civilization Community has worked hard, and they have created many great modifications for the game.  Unfortunately not all of these mods are compatible with each other, and require a lot of time and effort to merge these mods together, without causing errors.

The BUG Mod combines many of the best mods created for Civilization IV: Beyond the Sword, into one complete package. It also gives you the ability to pick which of these mods you would like to load, or disable, directly from the game.  In many cases we have taken the original mod, and have expanded it as well. 

 

 

-The BUG Mod for Beyond the Sword version 3.13

-The BUG Mod for Beyond the Sword Version 3.00 - 3.03

 

Installation Instructions:

(Warning: This will overwrite any mods currently in place. It’s recommended to backup your old CustomAssets folder. Directions to avoid this are given in the install program, or in The BUG Mod Help.chm if you downloaded the ZIP archive. Also, the BUG Mod.ini file MUST be in the root Beyond the Sword folder, or the options tab will cause the game to lock up.)

Self-Extracting .EXE instructions:
1)Run ‘BUGMod_XXX.exe’ file. (XXX will be replaced by the current version number.)
2)When you are prompted for the directory, if the one listed is not correct, browse for your CIV:BTS Documents folder.
3)Restart CIV.

ZIP File instructions:
1) Locate your CIV:BTS Document folder (Usually within My Documents/My Games/Beyond the Sword or Users/Username/Documents/My Games/Beyond the Sword for Vista).
2) Copy contents of 'BUGMod_XXX.zip’ file, (XXX will be replaced by the current version number.), into the CIV:BTS Documents folder. Make sure BUG.INI is placed in the Beyond The Sword folder, not the CustomAssets folder.
3) Restart CIV.
