Options Screen

The core to The BUG Mod is the ability to pick and choose the way your game looks and plays. Once the game starts you can customize the mod by hitting Ctrl-Alt-O. From this screen you can adjust the look and feel of many of the features within the game.

 

Note: The Asioasioasio’s Wide Citybar mod can’t be removed using the Option Screen (Alt-Ctrl-O). If you want to remove it, simply delete the "CustomAssets\Art\Interface\Citybar" folder.
