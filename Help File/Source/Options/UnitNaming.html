Unit Naming Options

Enabled:

This will Enable the Unit Naming mod.

Use Advanced Methods:

When this option is checked, additional options are made avaliable to you, beyond what is given in The BUG Mod Options screen. Editing of the "Adv Unit Naming.ini" file will be required to go beyond the conventions used within the Options screen.

Naming Conventions:

You can configure the default naming convention on the first line. To use this for each combat type, simply type the word DEFAULT. You can also configure a naming convention for each of the combat types listed here as well.

 

** Please see the Unit Naming section of the help file for more details information, and suggestions for strings to use. **
