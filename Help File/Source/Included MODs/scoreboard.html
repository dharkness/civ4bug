Advanced Scoreboard by Emperor Fool

About Advanced Scoreboard

This mod provides a fully customizable scoreboard as part of The BUG Mod. It integrates both TheLopez's Dead Civ Scoreboard and Porge's Attitude Icons as well (both are still optional). It allows you to specify exactly which pieces to show and in what order. It also adds the option to display icons when you can see which technologies your rivals are researching rather than their names.

 

Instructions

You can customize how this mod works using the BUG Options screen's (Ctrl-Alt-O) Scoreboard tab. The options are all on the far right column under "Advanced Layout". With the "Enabled" box checked, you can customize your scoreboard layout.

The "Display Order" edit box allows you to choose which items to display and in what order. Each item has a unique single-character code. The edit box has hover text detailing the available codes:

  • S The civ's score
  • C The civ's/leader's name (see "Display Name" option)
  • ? Shows a ? if you haven't met the civ yet (unmet teammates of teams you have met, I think)
  • W Shows a fist if you are at war with the civ
  • E Shows a spy icon if you have a positive espionage point ratio against the civ
  • P The civ's power ratio compared to you
  • T The tech the civ is researching (vassals, teammates and espionage)
  • U The number of research turns left (see T above)
  • N Shows the three-arrow icon when you are connected to the civ's trade network
  • B Shows a scroll icon when you have signed an open borders agreement with the civ
  • D Shows a similar scroll icon when you have signed a defensive pact with the civ
  • R The civ's state religion
  • A The civ's attitude toward you
  • * You are waiting for this civ to finish its turn
  • L The civ's network stats (ping)
  • O The network player is out-of-sync

These items only display for rivals.

These items only display for multiplayer rivals.

 

Additional Options

  • The "Left-Align Names" checkbox allows you to align the civ/leader names along their left edge instead of their right edge.
  • The "Technology Icons" checkbox shows rival's technology research choices using icons rather than names.
