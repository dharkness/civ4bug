Power Ratios by EmperorFool

 

About Power Ratios

This mod adds a power ratio to the scoreboard for any civ against which you have enough espionage points to perform the Demographics passive mission. It displays a ratio of your power rating to theirs (higher is better) using one of three colors. Along with the default color you can set two ratio/color combinations (good and bad). The good color is used when the ratio is equal to or greater than the good ratio and vice versa for the bad ratio/color.

 

Instructions

You can enable or disable and customize this mod using the BUG Options screen's (Ctrl-Alt-O) Scoreboard tab. The options are all in the center column of that tab. You can also change the default colors in this tab as well.
