Shortcut Commands in The BUG Project

Below are the commands you will need to utilize The BUG Mod the best. There are not many, so should be fairly easy to memorize them.

  • Alt + Ctrl + O - BUG Options screen.

  • Alt + M - Set a reMinder

  • Alt + L - Toggle autoLog

  • Alt + B - Create Battle dump

  • Alt + E - Add custom Entry to autolog.
