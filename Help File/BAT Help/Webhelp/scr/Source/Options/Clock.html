Clock Options

Enable NJAGC:

This will turn on the Not Just Another Game Clock Mod.

Display Era:

Will Display the area on the left hand side of the screen, next to the gold.

Use Era-Related Colors:

This will color code the name of each era displayed. You can also choose which color you want each era to display as by using the drop down menus next to each era name in the Options Screen.

 

Alternate Game Clock:

This toggles between the Standard View and Alternate View of the Game Clock. When unchecked, only Standard View will be shown.

Peroid:

The clock will change between the Standard View and the Alternate View for the amount of time set in this option. The default amount is 5 seconds.

 

Display Options:

You can toggle what you want displayed in both the Standard View and the Alternate View here as well. The options you can choose are:

  • Clock
  • Completed Turned
  • Total Turns
  • Completed Percentage
  • Game Date and Golden Age Turns
