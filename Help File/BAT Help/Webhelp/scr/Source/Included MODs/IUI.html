Improved Unit Icons by ruff_hi

 

About Improved Unit Icons

Improved Unit Icons improves the icons that represent the units in the currently selected tile. It is a slimmed down version of the mod called Plot List Enhancement by 12Monkeys.

It will add 2 features to the tiles:

1) It will add a blue border to each unit which has enough exp to be promoted.

2) It will tell you the current action the unit is doing. So if it's fortified you will see the word FORT on the bottom of the unit icon, a automated worker will say AUTO, etc...

 

 

Instructions

You can turn off each of these features from with the BUG Options Screen, within the General Tab. Uncheck the boxes next to Promotion Available, or Unit Actions to turn off each of the respective features.
