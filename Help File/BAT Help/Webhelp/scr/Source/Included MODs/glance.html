Glance Screen by ruff_hi

 

About Improved Glance Screen

With version 3.13 of Beyond the Sword, the Glance Screen within the Advisor Screen was removed for, "being to ugly." We felt the information given in this screen was vital, and so returned it to the Foreign Advisor screen, but improved. We realigned the grid, as well as adding the attitude icons, and color coding the modifiers. It should now be easier to see where you stand with your rivals.

 

Instructions

Go to the Foreign Advisor Screen from the Main Interface, and click on Glance.
