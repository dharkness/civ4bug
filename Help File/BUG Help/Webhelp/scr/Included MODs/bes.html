About Better Espionage Screen

This is an improved Espionage Screen. It allows for more civilizations to be shown within the Espionage Screen, as well as vital information being color coded.

 

Instructions

From within the BUG Options Screen you can turn off color coding for ratios, as well as revert back to the original version of the Espionage Screen.
