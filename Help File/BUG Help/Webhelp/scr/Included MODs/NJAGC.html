Not Just Another Game Clock by TheLopez

 

About Not Just Another Game Clock

Just as the mod title says, this is not just another game clock mod. It is much, much more. This mod provides you the flexibility to most if not all of its different features. There are 12 main variables and one additional variable per era that can be used to configure the Not Just Another Game Clock Mod display information. Those variables are are configurable within the Options Screen. You can view those variables in Clock Options.
