Logging Options

Enable Logging:

Will turn on the Auto logging system.

Silent Start:

Will start logging without a pop-up asking for name of the log file.

 

Color Coding:

Will color code the log file to make it easier to find key events.

 

4000BC is Turn:

Default is 0. Will fix the problem with the game thinking your pre-turn is the first turn. Can be set to 1 for game default. This will not add a turn to the game, just logs it properly.

Show IBT:

Log what occurs In Between Turns with the other civilizations in the game.

 

Use Default File Name:

Path: Default is My Docs\My Games\Beyond the Sword\Autolog. You can change this to any directory you'd like.

File:

This will use the Name you selected for yourself within Civilization, as the default file name.

Prefix:

This will be used when you use the Alt + E command to enter your own Entries within the autologger. For example, using your nickname on a forum will cause all Alt + E Entries to have your nickname before it.

Format Style:

Will put the logger into preset configurations. The options are:

  • Plain - Will format the logger into a plain .txt format.

  • HTML Tags - Will format the logger to use HTML code for the log entries.

  • Forum Tags, With " around color codes - Will format the logger to make it easier to post on forums that require " " around color codes.

  • Forum Tags, No " around color codes - Will format the logger to make it easier to post on forums that do not require " " around color codes.

** Please see the Autologger section of this help file for details on what you can log or not log. **
