Extended Colors by SimCutie

 

About Extended Colors

This Mod will give you additional color values to use while modifying Civ4 or BTS. It is not a standalone mod, and will do nothing but allow modders and players to change their color values for a more personalized feel.

Civ4 comes with a very limited selection of colors, and made it difficult for modders to create color values that were standardized between mods. It’s also a long and time consuming process to add new colors, and prone to errors.

This file should help eliminate most of these problems. There are more then 150 additional colors to choose from, and these can even be modified to suit your taste.

To see a list of the colors available, and the code to use with them, please see Extended Colors Table.

 

Instructions

To use while coding with Python or XML:

Wrap color name with a square bracket.

Here is an Python Example:

[code] str = localText.getText("[B][COLOR_W_ORANGE][/B]This is Orange"

 

To return to standard Civ4 white text:

Text.[COLOR_REVERT]Now it's white text again.", ())

 

Here is an XML Example:

<English>[COLOR_UNIT_TEXT]%s1_GPName[COLOR_REVERT] has been born in

[B][COLOR_X_LIGHTSKYBLUE][/B]%s2_CityName[COLOR_REVERT]!</English>

<French>Naissance de [COLOR_UNIT_TEXT]%s1_GPName[COLOR_REVERT] &#224;

[B][COLOR_X_LIGHTSKYBLUE][/B]%s2_CityName[COLOR_REVERT] !</French>
